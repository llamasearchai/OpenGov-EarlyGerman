## Next Steps

### For Users
1. Clone the repository:
   ```bash
   git clone https://github.com/llamasearchai/OpenGov-EarlyGerman.git
   cd OpenGov-EarlyGerman
   ```

2. Install and run:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -e .
   deutsch version
   ```

3. Start the API:
   ```bash
   uvicorn opengov_earlygerman.api.main:app --reload
   ```
